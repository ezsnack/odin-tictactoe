# odin-tictactoe
tic tac toe project for the odin project

## how to play
Players X and O take turns, starting with X. Select a cell using the Arrow keys (or WASD or HJKL) and confirm your choice using the Enter key or the Spacebar.

## info
The application uses curses to process user input and opens a new window to display the game. When the game is over, the window closes and the winner (or draw result) is announced.
