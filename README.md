# odin-tictactoe
tic tac toe project for the odin project

## how to play
Players X and O take turns, starting with X. Select a cell using the Arrow keys (or WASD or HJKL) and confirm your choice using the Enter key or the Spacebar. You can tell which player is currently active by looking at the corners of the window.

## info
The application uses curses for interactivity and opens a new window to display the game. When the game is over, the window closes and the winner (or draw result) is announced.
